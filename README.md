# DIGITAL-LITERACY-PROJECT
This project focuses on promoting awareness and understanding of digital literacy in today’s technology-driven world. Digital literacy is the ability to effectively and responsibly use digital devices, the internet, and online platforms for communication, learning, and daily activities.

Project Overview
This Digital Literacy Project was completed as part of the CSE0001 course. The aim of this project is to develop essential digital skills, including creating a professional online presence, using online platforms effectively, communicating professionally, and understanding cyber safety.
The project consists of five tasks, each focusing on a different aspect of digital literacy.
